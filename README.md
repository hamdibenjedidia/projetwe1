# Site personnel - Projet Programmation Web 1

## Lancer le site
- Ouvrez `index.html` dans votre navigateur.

## Arborescence
```
projet_web_hamdi/
  index.html
  parcours.html
  competences.html
  experiences.html
  evenements.html
  matieres.html
  passions.html
  quiz.html
  contact.html
  temoignage.html
  assets/
    css/style.css
    js/main.js
    js/quiz.js
    img/...
  rapport/
    rapport.md
```

## GitHub Pages (hébergement)
1. Créez un repo GitHub (ex: `site-personnel`).
2. Uploadez tous les fichiers.
3. Sur GitHub : **Settings → Pages**.
4. Source : **Deploy from a branch**.
5. Branch : `main` / folder : `/root`.
6. Attendez que GitHub vous donne le lien.

## Notes
- CSS simple, sans framework.
- Quiz en JavaScript (note + bonnes réponses).
- Formulaire contact : ouvre l'application mail (mailto).
